# MacToDo User Manual

## Installation
1. Open the `MacToDo.dmg` file.
2. Drag `MacToDo` app to your Applications folder (or run it directly).

## Features

### 1. Task Management
- **Add Task**: Click the "+" button in the toolbar. Enter a title. Optionally set a category and due date.
- **Complete Task**: Click the circle icon next to a task to mark it as complete.
- **Delete Task**: Right-click on a task (or swipe) to delete it.

### 2. Categories
- Use the sidebar to filter tasks by category (Work, Personal, Learning, etc.).
- "All" shows all tasks.

### 3. Reminders
- When creating a task, toggle "Set Due Date & Reminder".
- Pick a date and time.
- The app will notify you when the time comes (requires notification permission).

### 4. Search
- (Future Feature) Currently you can filter by category.

## Troubleshooting
- **Notifications not working**: Ensure you allowed notifications when prompted, or check System Settings > Notifications > MacToDo.
